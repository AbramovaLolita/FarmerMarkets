# Farmer Markets App
1. Установите [Docker Desktop](https://www.docker.com).
2. Скачайте проект и откройте терминал в папке проекта.
3. Выполните команду: `docker-compose up --build`
4. Откройте в браузере: `https://localhost:6901`
5. Создайте файл .env и добавьте туда свои данные (шаблон в файле .env.example)
